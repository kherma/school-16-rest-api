## API (GET)

> SECTION\_\_NAME can be: testimonials, seats or concerts

```
/api/SECTION__NAME - return all items
/api/SECTION__NAME/NUMBER - return one item specific id = NUMBER
/api/SECTION__NAME/random - return random
```
